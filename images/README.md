# Инструкция по добавлению логотипа

## 📋 Что нужно сделать:

1. **Конвертируйте logo.pdf в PNG**
   - Откройте logo.pdf в любом редакторе (Photoshop, Illustrator, онлайн конвертер)
   - Экспортируйте как PNG с прозрачным фоном
   - Рекомендуемый размер: 200-300px по высоте
   - Сохраните качество изображения высоким

2. **Сохраните файл**
   - Название файла: `logo.png`
   - Место: `images/logo.png`
   - Убедитесь что фон прозрачный (не белый)

3. **Готово!**
   - Логотип автоматически появится:
     - В навигационном меню (слева)
     - В футере
     - Как favicon (иконка вкладки браузера)

## 🛠️ Онлайн конвертеры PDF → PNG:

- https://www.ilovepdf.com/pdf_to_jpg
- https://cloudconvert.com/pdf-to-png
- https://www.adobe.com/acrobat/online/pdf-to-jpg.html

## ⚙️ Если нужно изменить размер логотипа:

В файле `styles.css`:
- Навигация: `.nav-brand .logo { height: 50px; }` (строка ~79)
- Футер: `.footer-logo { height: 60px; }` (строка ~725)
